### karesansui-u

日本のエンタープライズ領域で大規模システムの設計・開発・品質を担当しているソフトウェアエンジニアです。
レガシー環境での開発経験が長いですが、最近は LLM を活用した開発の可能性に興味を持ち、日々勉強しています。

業務ではソフトウェア品質とアーキテクチャに関心を持って取り組んでいます。
OSS のコードを読む機会が増え、気になった箇所があれば PR を送るようにしています。
メンテナの方の負担にならないよう、確信のある修正だけを丁寧に出すことを心がけています。

#### 関心領域
- ソフトウェア品質・テスタビリティ
- システムアーキテクチャ
- 開発者ツール

#### 趣味
漫画・アニメが好きです。ヒロアカ / 葬送のフリーレン / 呪術廻戦 をよく読んでいます。
